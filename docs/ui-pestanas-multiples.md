# Uso de múltiples herramientas en pestañas

## Descripción

Escuadra permite trabajar con varias herramientas simultáneamente mediante pestañas independientes dentro de la interfaz gráfica.

Esta funcionalidad facilita realizar distintos cálculos al mismo tiempo, comparar resultados y mantener varias herramientas abiertas sin perder el contexto de trabajo.

---

## ¿Qué son las pestañas?

Las pestañas funcionan de manera similar a las de un navegador web. Cada pestaña representa una herramienta abierta dentro de la aplicación y permite cambiar rápidamente entre distintas tareas.

### Beneficios

| Beneficio | Descripción |
|-----------|-------------|
| Multitarea | Permite trabajar con varias herramientas al mismo tiempo |
| Comparación | Facilita comparar resultados entre distintos cálculos |
| Organización | Mantiene separadas las herramientas abiertas |
| Continuidad | Conserva la información visible mientras la pestaña permanezca abierta |

---

## Abrir herramientas en pestañas

Cuando una herramienta se abre desde la interfaz de Escuadra, esta puede mostrarse en una pestaña independiente dentro del área de trabajo.

Esto permite tener varias herramientas disponibles simultáneamente y alternar entre ellas según sea necesario.

---

## Cambiar entre pestañas

Para navegar entre herramientas abiertas:

1. Ubica la barra de pestañas de la aplicación.
2. Selecciona la pestaña correspondiente a la herramienta que deseas utilizar.
3. La interfaz mostrará inmediatamente la herramienta seleccionada.

---

## Trabajar con múltiples herramientas

Las pestañas permiten mantener diferentes cálculos abiertos al mismo tiempo.

Por ejemplo:

- Conversión de unidades.
- Ley de Ohm.
- Sistemas lineales.
- Herramientas geométricas.
- Herramientas estructurales.

Cada pestaña puede utilizarse de forma independiente, facilitando la consulta de resultados previos mientras se realizan nuevos cálculos.

---

## Cerrar una pestaña

Cuando una herramienta deja de ser necesaria, su pestaña puede cerrarse desde la interfaz de la aplicación.

Se recomienda revisar los resultados antes de cerrar una pestaña para evitar perder información que aún sea necesaria para el trabajo actual.

---

## Ejemplo de uso

Un estudiante puede mantener abiertas simultáneamente varias herramientas mientras resuelve un ejercicio:

1. Una pestaña para cálculos de vigas.
2. Una pestaña para conversión de unidades.
3. Una pestaña para cálculos geométricos.

De esta manera puede consultar resultados entre herramientas sin necesidad de abrirlas nuevamente.

---

## Recomendaciones

- Mantener abiertas únicamente las herramientas necesarias.
- Utilizar pestañas diferentes para cálculos relacionados.
- Revisar los resultados antes de cerrar una pestaña.
- Organizar las herramientas según el flujo de trabajo que se esté realizando.
