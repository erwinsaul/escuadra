# Requerimientos del Sistema

## Requerimientos Funcionales (RF)

| ID     | Descripción                                                                                                                      | Prioridad |
| ------ | -------------------------------------------------------------------------------------------------------------------------------- | --------- |
| RF-001 | El sistema debe permitir realizar cálculos geométricos de áreas para diferentes figuras.                                         | Alta      |
| RF-002 | El sistema debe permitir calcular perímetros de figuras geométricas soportadas.                                                  | Alta      |
| RF-003 | El sistema debe permitir calcular volúmenes de cuerpos geométricos soportados.                                                   | Alta      |
| RF-004 | El sistema debe permitir realizar conversiones de temperatura entre distintas escalas.                                           | Alta      |
| RF-005 | El sistema debe permitir realizar conversiones de unidades de longitud.                                                          | Alta      |
| RF-006 | El sistema debe permitir realizar conversiones de unidades de masa.                                                              | Media     |
| RF-007 | El sistema debe permitir realizar conversiones de velocidad, energía y presión.                                                  | Media     |
| RF-008 | El sistema debe permitir resolver cálculos eléctricos utilizando la Ley de Ohm.                                                  | Alta      |
| RF-009 | El sistema debe permitir calcular potencia eléctrica, divisores de tensión y circuitos serie/paralelo.                           | Alta      |
| RF-010 | El sistema debe permitir realizar operaciones matemáticas avanzadas como trigonometría, estadística y potencias.                 | Alta      |
| RF-011 | El sistema debe permitir resolver sistemas de ecuaciones lineales.                                                               | Media     |
| RF-012 | El sistema debe permitir realizar cálculos de ingeniería civil relacionados con vigas, cargas distribuidas y momentos flectores. | Media     |
| RF-013 | El sistema debe permitir realizar conversiones de bases numéricas y cálculos relacionados con sistemas digitales.                | Media     |
| RF-014 | El sistema debe permitir generar tablas de verdad y consultar códigos ASCII.                                                     | Baja      |
| RF-015 | El sistema debe permitir ejecutar las herramientas disponibles mediante una interfaz de línea de comandos (CLI).                 | Alta      |

## Requerimientos No Funcionales (RNF)

| ID      | Categoría      | Descripción                                                                                              | Prioridad |
| ------- | -------------- | -------------------------------------------------------------------------------------------------------- | --------- |
| RNF-001 | Mantenibilidad | El código debe ser compatible con Python 3.8 o versiones superiores.                                     | Alta      |
| RNF-002 | Mantenibilidad | La documentación técnica debe mantenerse actualizada con cada nueva funcionalidad incorporada.           | Media     |
| RNF-003 | Mantenibilidad | El proyecto debe mantener una estructura modular para facilitar la incorporación de nuevas herramientas. | Alta      |
| RNF-004 | Mantenibilidad | El código debe seguir estándares de estilo y buenas prácticas de desarrollo.                             | Media     |
| RNF-005 | Mantenibilidad | El sistema debe contar con pruebas automatizadas para validar las funcionalidades principales.           | Alta      |
| RNF-006 | Rendimiento    | Las operaciones de cálculo deben completarse en menos de un segundo para entradas normales.              | Alta      |
| RNF-007 | Rendimiento    | El consumo de memoria debe mantenerse estable durante la ejecución de cálculos complejos.                | Media     |
| RNF-008 | Usabilidad     | La aplicación debe poder ejecutarse desde una interfaz de línea de comandos intuitiva.                   | Alta      |
| RNF-009 | Usabilidad     | Los resultados de los cálculos deben presentarse de forma legible y consistente.                         | Alta      |
| RNF-010 | Usabilidad     | Los mensajes de error deben ser claros y comprensibles para el usuario.                                  | Media     |
