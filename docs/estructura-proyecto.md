# Estructura del Proyecto

Este documento describe la arquitectura de directorios del repositorio para orientar a nuevos contribuidores y facilitar la navegación por el código.

## Árbol de Directorios

```text
.
├── .github/          # Flujos de trabajo de GitHub Actions
├── data/             # Archivos de datos, datasets y recursos crudos
├── docs/             # Documentación del proyecto
├── security/         # Configuraciones y herramientas de seguridad
├── src/              # Código fuente principal (Python)
└── tests/            # Pruebas unitarias y de integración
