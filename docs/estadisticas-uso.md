# Estadísticas de uso de herramientas

Este documento describe el panel de estadísticas de uso de herramientas de la suite Escuadra. El panel permite visualizar qué herramientas se han utilizado, con qué frecuencia y cómo se distribuye ese uso según la carrera del usuario.

## ¿Qué muestra el panel?

El panel de estadísticas presenta un resumen agregado del uso de herramientas, organizado por carrera. Para cada carrera se indica:

- Cuántas veces fue usada cada herramienta.
- Qué porcentaje representa cada herramienta sobre el total de usos de esa carrera.
- Cuál es la herramienta más utilizada.

## Actores que generan estadísticas

- **Estudiante de ingeniería:** principal generador de uso, utiliza las herramientas para resolver ejercicios y verificar cálculos.
- **Docente:** genera uso al demostrar procedimientos y plantear ejemplos prácticos en clase.

## Notas

- Los datos de uso se recopilan de forma anónima durante las sesiones activas.
- El panel se actualiza en tiempo real a medida que los usuarios interactúan con las herramientas.
- Los porcentajes se calculan sobre el total de usos registrados por carrera, no sobre el total global.
- Las carreras mostradas corresponden a las registradas en el sistema al momento de consulta.
