# 📦 Modo Batch del CLI

Esta guía explica cómo utilizar el **modo batch** de Escuadra, una funcionalidad que permite procesar múltiples cálculos desde un archivo de entrada.

---

## 🎯 ¿Qué es el modo batch?

El modo batch permite **ejecutar varios cálculos de forma automatizada** a partir de un archivo de entrada (CSV o JSON). Es útil para:

- **Procesar grandes volúmenes de datos**.
- **Automatizar cálculos repetitivos**.
- **Generar reportes en lote**.
- **Integrar** con otros sistemas.

---

## 🚀 Cómo usar el modo batch

### Comando principal

```bash
escuadra batch --archivo entrada.csv --salida resultados.csv
